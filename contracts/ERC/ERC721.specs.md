# ERC721.sol User Stories

## Scope
Non-fungible token standard for unique ownership, transfers, and approvals.

## Primary Actors
- NFT owner
- Approved operator
- Marketplace/integrator

## User Stories
- As an NFT owner, I want to transfer my token to another address, so that I can trade or gift it.
- As an NFT owner, I want to approve an address for a specific token, so that marketplaces can execute sales.
- As an NFT owner, I want to approve an operator for all my NFTs, so that portfolio-level management is easy.
- As an integrator, I want owner and token metadata query functions, so that wallet and listing displays are accurate.

## Acceptance Criteria
- Transfers require caller authorization and valid ownership.
- Safe transfers check receiver interface support to prevent accidental token loss.
- Approvals are reset appropriately after transfers.
- Required ERC721 events are emitted for state changes.
